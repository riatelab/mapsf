

# Package description

## Description

Create and integrate thematic maps in your workflow. This package helps
to design various cartographic representations such as proportional
symbols, choropleth or typology maps. It also offers several functions
to display layout elements that improve the graphic presentation of maps
(e.g. scale bar, north arrow, title, labels). <code>mapsf</code> maps
<code>sf</code> objects on <code>base</code> graphics.

A "Get Started" <strong>vignette</strong> contains commented scripts on
how to create various maps: <code>vignette(topic = “mapsf”, package =
“mapsf”)</code>

## Symbology

These functions display cartographic layers.

<ul>
<li>

<code>mf_map()</code> Plot a map

</li>
<li>

<code>mf_label()</code> Plot labels

</li>
<li>

<code>mf_raster()</code> Plot a raster

</li>
<li>

<code>mf_graticule()</code> Plot graticules

</li>
</ul>

## Map layout

These functions are dedicated to the map layout design.

<ul>
<li>

<code>mf_theme()</code> Set a theme

</li>
<li>

<code>mf_shadow()</code> Plot a shadow

</li>
<li>

<code>mf_background()</code> Plot a background image

</li>
<li>

<code>mf_annotation()</code> Plot an annotation

</li>
<li>

<code>mf_arrow()</code> Plot a north arrow

</li>
<li>

<code>mf_credits()</code> Plot credits

</li>
<li>

<code>mf_layout()</code> Plot a map layout

</li>
<li>

<code>mf_title()</code> Plot a title

</li>
<li>

<code>mf_scale()</code> Plot a scale bar

</li>
<li>

<code>mf_inset_on()</code> / <code>mf_inset_off()</code> Plot an inset

</li>
<li>

<code>mf_worldmap()</code> Plot a point on a world map

</li>
<li>

<code>mf_legend()</code> Plot a legend

</li>
</ul>

## Utility functions

<ul>
<li>

<code>mf_svg()</code> Export a map in SVG file format

</li>
<li>

<code>mf_png()</code> Export a map in SVG file format

</li>
<li>

<code>mf_distr()</code> Plot a distribution

</li>
<li>

<code>mf_get_links()</code> Get a link layer from a data.frame of links

</li>
<li>

<code>mf_get_pal()</code> Get color palettes

</li>
<li>

<code>mf_get_breaks()</code> Get class intervals

</li>
<li>

<code>mf_get_mtq()</code> Get the ‘mtq’ dataset

</li>
<li>

<code>mf_get_ratio()</code> Get map width and height values

</li>
<li>

<code>mf_get_pencil()</code> Get a pencil layer from polygons

</li>
<li>

<code>mf_get_borders()</code> Get a border layer from polygons
